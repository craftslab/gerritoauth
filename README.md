Gerrit OAuth2 authentication provider
=====================================

This repository contains the OAuth plugin for two Gerrit release lines:

* Gerrit 2.13 plugin source in [2.13](2.13)
* Gerrit 3.4 plugin source in [3.4](3.4)

Pick the matching directory for your Gerrit version and follow the build
and install instructions below.

Supported providers
-------------------

Gerrit 2.13 (from [2.13](2.13)):

* Bitbucket
* CAS
* GitHub
* Google

Gerrit 3.4 (from [3.4](3.4)):

* AirVantage
* Bitbucket
* CAS
* CoreOS Dex
* Facebook
* GitHub
* GitLab
* Google
* Keycloak
* LemonLDAP::NG
* Azure (previously Office365)
* Phabricator
